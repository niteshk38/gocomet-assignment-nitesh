# GoComet-Assignment Nitesh Kumar

This repository contains a UI automation framework built as part of an **GoComet-Assignment**.  
The framework automates the **Login** of a sample web application and is integrated with **CI/CD** for automatic execution.

---

## Application Under Test
https://the-internet.herokuapp.com/login

---

## Tech Stack
- Java 17
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager
- GitHub Actions (CI/CD)

---

## TestCases

• Verify successful login with valid username and password.
• Verify login failure with appropriate error message for invalid credentials.

## Clone Repository

git clone https://github.com/niteshk38/gocomet-assignment-nitesh
